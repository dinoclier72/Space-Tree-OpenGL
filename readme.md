Projet de sintheses d'images qui a pour but de créer une scène 3D avec un rendu réaliste qui utilise des textures et des lumières.
# installation
utiliser un environnement virtuel python
```bash
python -m venv .venv
./.venv/bin/activate
pip install -r constraints.txt
python projet.py
```
